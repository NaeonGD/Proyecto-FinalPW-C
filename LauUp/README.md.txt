# ✦ Lau & Up. — Tienda de Belleza
**Universidad Surcolombiana · Programación Web**

## Requisitos
- Java 21
- MySQL 8
- Maven

## Instalación

1. Clonar el repositorio
2. Crear la base de datos ejecutando `database.sql` en MySQL
3. Copiar `src/main/resources/application.properties.example` 
   como `src/main/resources/application.properties`
4. Rellenar las credenciales en `application.properties`
5. Crear el certificado SSL con Keystore Explorer y guardarlo
   como `src/main/resources/lauup.jks`
6. Importar en Eclipse como Maven Project
7. Ejecutar `LauUpApplication.java`
8. Abrir `https://localhost:8443`
